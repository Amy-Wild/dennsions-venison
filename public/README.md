# Images Folder

Upload your images here:

## Required Images:
- `logo.png` - Your company logo
- `hero-bg.jpg` - Main hero background image
- `shoot-1.jpg`, `shoot-2.jpg` - Hunting/countryside photos
- `products-1.jpg`, `products-2.jpg` - Venison product photos
- `favicon.ico` - Website favicon

## Image Guidelines:
- Use JPG for photos (smaller file size)
- Use PNG for logos with transparency
- Optimize images before uploading (max 1-2MB per image)
- Recommended hero image size: 1920x1080px
- Recommended product images: 800x600px
