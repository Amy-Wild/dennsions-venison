# Dennison's Venison

Wild, Honest, Fair - Premium venison and deer management services from the UK countryside.

## Website

Visit us at: [www.dennisonsvenison.co.uk](https://www.dennisonsvenison.co.uk)

## About

Founded by Lee Dennison, Dennison's Venison provides:
- Premium wild venison products
- Professional deer management services
- Ethical, sustainable, traceable meat from hill to plate

Based at Bradshaw Trees Farm, Mellor, Stockport.

## Development

This is a Next.js static site optimized for SEO and fast loading.

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Images

Add your images to the `public/` folder:
- `public/logo.png` - Company logo
- `public/hero-bg.jpg` - Hero background image
- `public/shoot-*.jpg` - Hunting/countryside photos
- `public/products-*.jpg` - Venison product photos
